# 图片推荐清单

来源：`用户手册/（对外）Drive CO1.6-用户手册.docx`

说明：

- 本清单面向当前四页官网文案：工作台、协助空间、知识库、企业管理
- 已从用户手册中提取并筛选出一批高相关截图，放在当前目录
- 推荐分为：
  - `首选`：和当前文案最匹配，可作为官网配图主候选
  - `备选`：功能相关，但更适合作为辅图或模块图
  - `不建议直接上线`：适合做设计参考，不建议直接用于官网

## 一、整体结论

### 1. 最成熟的是知识库图片

知识库在手册中有完整的集合页、内容页、设置页、公开范围、成员权限、文件安全等截图，最适合直接支撑官网页面。

### 2. 协助空间图片基本够用，但要注意旧命名

协助空间的截图来自手册中“团队空间”章节，功能是对的，但界面文字仍是旧称“团队空间”。如果官网要统一为“协助空间”，建议优先重新截图，或在使用前确认是否需要重拍。

### 3. 工作台缺少真正意义上的“官网首屏图”

手册中更多是“最近文件”“我的桌面”等页面截图，而不是一个强烈体现“统一入口”的综合工作台页面。因此工作台更适合用这些图做功能模块图，不建议直接拿它们做官网 hero 图。

### 4. 企业管理图片不够完整

目前从手册里明确提取到最适合官网使用的是“通讯录”截图；“操作日志”“企业设置”“企业回收站”等更贴近当前文案的图，在这份手册里没有找到同等完整度的截图，建议补拍。

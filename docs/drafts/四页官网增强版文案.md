# 四页官网增强版文案

适用目标：

- 在“事实准确”的基础上，提升官网页面的吸引力、记忆点和转化感
- 保留产品真实能力边界，不使用无法被演示或交付承接的夸张表述
- 适合用于官网产品介绍页、市场页、销售线索承接页

对应页面：

- `portal.html` 工作台
- `space.html` 协助空间
- `knowledge-base.html` 知识库
- `management-panel.html` 企业管理

## 一、官网增强版写法原则

### 1. 首屏不只讲“是什么”，更要讲“为什么值得现在看”

官网首屏建议采用三段式：

- 这是什么
- 带来什么结果
- 为什么和别家不一样

### 2. 优先写客户感知得到的结果

比起“统一、承接、衔接、聚合”这类抽象词，官网更适合直接写：

- 减少信息遗漏
- 让资料不再散落
- 降低跨团队协作成本
- 让制度真正被找到和使用
- 让权限和安全规则真正落地

### 3. 每页都要有自己的“主卖点”

- 工作台：每天打开就能开始工作
- 协助空间：把项目协作放进同一个空间
- 知识库：把沉淀内容变成可复用的企业资产
- 企业管理：让组织、权限与安全策略统一落地

### 4. CTA 不能只写“预约演示”

建议每页使用更贴近主题的行动按钮文案，增强点击动机。
